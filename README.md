# Personal Portfolio Website

This is my personal portfolio website built using HTML and CSS. It showcases my projects, skills, and background as a junior developer.

## Description

The website includes an "About Me" section, a projects section with images and descriptions, and a contact section with links to my GitHub and email. It is designed with a clean layout, a soft pink color scheme, and responsive styling to ensure it looks good on both desktop and mobile devices.

## Features

* Responsive layout using CSS Grid and Flexbox
* Project cards with images and descriptions
* Hover effects for interactivity
* Styled links and consistent color scheme
* Organized and readable HTML/CSS structure
